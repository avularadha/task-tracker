# 🚀 TaskTracker

A modern and responsive Task Management Application built using **React + Vite**.

---

## 🌐 Live Demo

👉 https://task-tracker-qry4.vercel.app

---

## ✨ Features

- ➕ Add Tasks
- 🔍 Search Tasks
- 🌙 Light / Dark Mode
- 📊 Task Statistics
  - Total Tasks
  - Completed Tasks
  - Pending Tasks
- 🗑️ Clear All Tasks
- 📱 Responsive Design
- ⚡ Fast Performance

---

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript
- CSS3

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/avularadha/task-tracker.git
```

Move into project folder:

```bash
cd tasktracker
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
